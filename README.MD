# Location-Based Incident Management System

A web-based system for reporting, assigning, tracking, and monitoring incidents based on location.

## Tech Stack

- Frontend: React + Tailwind CSS
- Backend: Node.js + Express.js
- Database: MySQL
- Map: Leaflet + OpenStreetMap
- Authentication: JWT
